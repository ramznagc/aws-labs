# 🌐 IP Addressing Cheat Sheet

A quick reference guide for IPv4 addressing concepts.

---

## 📌 IPv4 Address Classes

| Class | First Octet Range | Network / Host |
|--------|-------------------|----------------|
| A | 1 – 126 | N.H.H.H |
| B | 128 – 191 | N.N.H.H |
| C | 192 – 223 | N.N.N.H |
| D | 224 – 239 | Multicast |
| E | 240 – 255 | Reserved |

---

## 🔒 Private IP Address Ranges

- **10.0.0.0 – 10.255.255.255**
- **172.16.0.0 – 172.31.255.255**
- **192.168.0.0 – 192.168.255.255**

---

## 🌍 Common Special Addresses

| Address | Description |
|---------|-------------|
| 127.0.0.1 | Loopback |
| 0.0.0.0 | Default / Unspecified |
| 255.255.255.255 | Broadcast |

---

## 🔄 NAT Types

- Static NAT (SNAT)
- Dynamic NAT (DNAT)
- Port Address Translation (PAT)

---

## 🚀 IPv6 Highlights

- 128-bit addressing
- Huge address space
- No broadcast
- Better routing efficiency
